- 주기적으로 올라오는 개발자 채용공고를 한눈에 보기 위해서 제작하였습니다.

## 실행 방법

1. `git clone`
2. `npm install`
3. `npm run dev`

## 기술

- Next.js
- TypeScript
- Emotion
- Cheerio
